# Personal Portfolio Website

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

This repository contains a **responsive personal portfolio website** developed using **HTML and CSS**.  
The website is designed to showcase personal details, educational background, projects, and contact information in a clean and professional layout.

---

## 📌 Features

- Fixed navigation bar with smooth scrolling
- Full-screen hero (home) section
- Education section with academic details
- Projects section with GitHub repository links
- Contact section with personal contact details
- Responsive design for mobile, tablet, and desktop screens
- CSS hover effects and animations
- Built using **HTML5 and CSS3 only**

---

## 🛠️ Technologies Used

- **HTML5** – Website structure
- **CSS3** – Styling, layout, responsiveness, and animations
